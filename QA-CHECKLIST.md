# QA Checklist

## Portfolio Requirements
- [ ] Site is live and accessible via public URL
- [ ] Site is fully responsive on mobile and desktop
- [ ] At least 4 projects are showcased
- [ ] Each project is appropriately displayed and described
- [ ] Work samples are organized with thumbnails and clear navigation
- [ ] Home page includes featured project and gives a sense of authorship/personality
- [ ] About page includes personal background and practicum intentions
- [ ] Each page has consistent navigation
- [ ] Each page includes name and email contact link
- [ ] Overall design is professional and visually consistent

## Browser Support
- [ ] Chrome (latest)
- [ ] Firefox (latest)
- [ ] Safari (latest)
- [ ] Edge (latest)

## Devices
- [ ] Mobile – iOS (375 x 812)
- [ ] Mobile – Android (360 x 800)
- [ ] Tablet (768 x 1024)
- [ ] Laptop (1366 x 768)
- [ ] Desktop (1920 x 1080)
- [ ] Large Desktop (2560 x 1440)

## Code & Optimization
- [ ] No errors in browser developer console on any page
- [ ] All pages pass W3C HTML Validator
- [ ] PageSpeed Insights – Mobile 50+
- [ ] PageSpeed Insights – Desktop 75+
- [ ] All pages under 2-4 MB depending on size
- [ ] CSS and JS minified in production build
- [ ] No dead/broken links on any page (https://www.deadlinkchecker.com/)
- [ ] 404 page exists for unknown routes
- [ ] No `console.log` statements left in production code

## CMS
- [ ] N/A

## Metadata & SEO
- [ ] Unique `<title>` tag on every page
- [ ] Meta description present on every page
- [ ] Open Graph tags present (`og:title`, `og:description`, `og:image`)
- [ ] Favicon displays correctly in browser tab
- [ ] All images have descriptive `alt` text

## Content Accuracy & Quality
- [ ] No spelling or grammar errors on any page
- [ ] All project descriptions are complete and accurate
- [ ] No placeholder text (e.g. "Lorem ipsum") remaining
- [ ] Resume download link works and file is up to date

## Assets & Media
- [ ] All images load correctly (no broken image icons)
- [ ] Images are appropriate and correctly matched to content
- [ ] All images are appropriately resized (under 500kb)
- [ ] No copyrighted material used without permission

## Accessibility
- [ ] Color contrast meets WCAG AA standard
- [ ] Focus states are visible on all interactive elements
